# Ultimate Exam Arena

This repository is a Node.js + Express backend with a static vanilla HTML/CSS/JS frontend.

Primary goals:
- Keep changes small and focused on the requested feature.
- Preserve existing app structure and design patterns.
- Fix bugs or add UI improvements only where needed.
- Avoid introducing large new frameworks or unnecessary dependencies.

The main areas of work are:
- `backend/` for auth, admin, exams, dashboard, and API routes.
- `frontend/` for vanilla page markup, styles, and client-side scripts.

When editing, prefer clarity, maintainability, and compatibility with the current static site layout.
